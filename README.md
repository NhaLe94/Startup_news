# Project Nodejs-api-security

## Việc cần làm sau khi clone:
1. Cài gulp-cli `npm install --global gulp-cli`
2. chạy `npm install` để cài tất cả dependencies đã khai báo trong package.json

## Để chạy
`npm start` hoặc `gulp` để chạy
